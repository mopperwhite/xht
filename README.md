
# XHentai

A local doujinshi reader.

# Usage

## Start Server

1. Clone this project.

2. Install dependencies.

```bash
bundle install
```

3. Start web server.

```bash
rackup -p 8000
```

4. Read doujinshi at [http://localhost:8000/](http://127.0.0.1:8000/)


## Set authorization

```bash
rake seed
```
